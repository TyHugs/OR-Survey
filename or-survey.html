One-time setup — connect to Google Sheets

Step 1. Create a new Google Sheet (name it anything, e.g., "OR Survey Responses").
Step 2. In the Sheet, go to Extensions → Apps Script.
Step 3. Delete any code in the editor and paste the Apps Script code (provided separately).
Step 4. Click Deploy → New deployment. Set type = Web app, Execute as = Me, Who has access = Anyone. Click Deploy.
Step 5. Copy the Web app URL. Open this HTML file and paste it where it says APPS_SCRIPT_URL near the top of the <script> section.
Step 6. Push to GitHub. Done — every submission flows into your Sheet automatically.
